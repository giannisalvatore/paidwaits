# WaitingAds — Get paid for waiting

Trasforma l'attesa di Claude Code in guadagno: durante il "thinking" l'estensione
mostra una pubblicità nativa sopra lo spinner, e il **50%** del ricavo delle
impression va a te.

## Comandi
- **Paidwaits: Connect account** — login e attivazione su tutti i target trovati
  (Claude Code webview, Claude CLI statusline, Codex, Codex CLI).
- **Paidwaits: Restore Claude Code** — rimuove la patch e ripristina i file originali.
- **Paidwaits: Earnings status** — guadagni e target attivi.
- **Paidwaits: Diagnose** — stato di ogni target (versione, compatibilità, patch).

## Privacy
L'estensione **non** legge il contenuto delle tue conversazioni con Claude: rileva
solo quando Claude sta elaborando, per mostrare l'ad. La patch è completamente
reversibile (backup byte-exact). Killswitch lato server per disattivazione remota.
